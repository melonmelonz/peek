# PEEK Web Implementation Plan

**Goal:** Ship a wasm32 build of PEEK that runs full-fidelity in the browser
and embeds inside `goolz.org/next-chapter` via iframe.

**Architecture:** New `peek-web` cdylib crate uses ratzilla DomBackend.
A `Storage` trait extracted into `peek-core` lets native use `FsStorage` and
web use `LocalStorage`. A `SceneRunner` extracted from `peek-cli` is shared
by both targets.

**Tech Stack:** Rust 2021, ratzilla, ratatui 0.27, wasm-bindgen, web-sys,
js-sys, chrono with `wasmbind`, trunk for builds.

---

## Task 1: Storage trait in peek-core

**Files:**
- Create: `peek-core/src/storage.rs`
- Modify: `peek-core/src/lib.rs` (re-export)
- Modify: `peek-core/src/state.rs`, `peek-core/src/memorial.rs` (move FS code into trait impl)

Define `Storage` trait + `FsStorage` impl. Existing call sites in
`peek-cli` move from free functions to `FsStorage::new(state_path, memorial_path).save_state(...)`.

## Task 2: Input event normalization in peek-tui

**Files:**
- Create: `peek-tui/src/input.rs` with `InputEvent` enum and `From<crossterm::event::Event>`
- Modify: `peek-tui/src/scene.rs` — `Scene::handle` takes `&InputEvent`
- Modify all `peek-tui/src/scenes/*.rs` to match on `InputEvent` variants

`InputEvent` covers: `Key(KeyCode)` (a-z, A-Z, digits, special), `Quit`, `Tick`.

## Task 3: SceneRunner in peek-tui

**Files:**
- Create: `peek-tui/src/scene_runner.rs`
- Modify: `peek-tui/src/lib.rs` (re-export)
- Modify: `peek-cli/src/cmd/tui.rs` to use SceneRunner

`SceneRunner<S: Storage>` owns the App, current scene, and scene transitions.
Methods: `handle_input`, `tick`, `render(terminal)`. Returns `RunnerOutcome::Continue | Quit`.

## Task 4: peek-web crate skeleton

**Files:**
- Create: `peek-web/Cargo.toml`, `peek-web/src/lib.rs`, `peek-web/index.html`,
  `peek-web/style.css`, `peek-web/Trunk.toml`
- Modify: `Cargo.toml` (add member)

`#[wasm_bindgen(start)] fn start()` builds a ratzilla DomBackend, constructs
a `SceneRunner` with `LocalStorage`, attaches a key event listener, kicks
off the rAF loop.

## Task 5: LocalStorage impl

**Files:**
- Create: `peek-web/src/storage.rs`

`LocalStorage` impl of `peek_core::Storage` using `web_sys::window().local_storage()`.
Keys: `peek/state`, `peek/memorials`. Same RON serde format.

## Task 6: Web input adapter

**Files:**
- Create: `peek-web/src/input.rs`

Convert ratzilla `KeyEvent` to `peek_tui::InputEvent`.

## Task 7: Web runtime loop

**Files:**
- Create: `peek-web/src/runtime.rs`

Wire ratzilla `WebRenderer::on_key_event` and a 100ms `setInterval`-style
tick to `SceneRunner`. Render via `terminal.draw_web(|frame| runner.render(frame))`.
